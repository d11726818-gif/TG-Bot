🤖 Private Feedback Bot
Анонимный Telegram-бот для обратной связи с администратором. Позволяет пользователям отправлять сообщения, сохраняя визуальную анонимность, и участвовать в рейтинге активности.
✨ Особенности
Анонимные сообщения: Администратор получает сообщения без ссылки на профиль отправителя.
Двусторонняя связь: Администратор может отвечать на сообщения через функцию "Reply" (Ответ).
Статус "Прочитано": Пользователь получает уведомление, когда администратор увидел сообщение.
Топ активных: Публичный рейтинг пользователей по количеству отправленных сообщений.
Управление приватностью: Каждый пользователь может скрыть свой ник из ТОПа одной кнопкой.
Поддержка медиа: Бот корректно пересылает фото и видео без раскрытия данных автора.
🛡 Анонимность и безопасность
Для пользователя: Бот скрывает ваш профиль в чате с администратором. Однако для работы системы ответов и ТОПа бот сохраняет ваш ID и Username/Name в локальную базу данных.
Для администратора: Вы не видите, кто вам пишет, пока не заглянете в базу данных. Это позволяет получать более честную и непредвзятую обратную связь.
Безопасность: Токен бота не хранится в коде, а используется через переменные окружения.
🚀 Запуск бота
1. Подготовка
Убедитесь, что у вас установлен Python 3.8+ и библиотека aiogram:
pip install aiogram
2. Настройка переменных
Бот использует переменную окружения BOT_TOKEN.
Вставьте свой ID в переменную ADMIN_ID внутри файла бота.
Windows:
set BOT_TOKEN=ваш_токен_из_BotFather
python main.py
Linux / macOS:
export BOT_TOKEN="ваш_токен_из_BotFather"
python main.py
🛠 Технологии
Язык: Python 3
Библиотека: aiogram 3.x
База данных: SQLite (локальный файл feedback_private.db)
FSM: Хранение состояний для удобного ввода сообщений.
