# 🤖 Private Feedback Bot

Анонимный Telegram-бот для обратной связи с администратором. Позволяет пользователям отправлять сообщения, сохраняя визуальную анонимность, и участвовать в рейтинге активности.

## ✨ Особенности

*   **Анонимные сообщения:** Администратор получает сообщения без ссылки на профиль отправителя.
*   **Двусторонняя связь:** Администратор может отвечать на сообщения через функцию "Reply" (Ответ).
*   **Статус "Прочитано":** Пользователь получает уведомление, когда администратор ответил на сообщение.
*   **Топ активных:** Публичный рейтинг пользователей по количеству отправленных сообщений.
*   **Управление приватностью:** Каждый пользователь может скрыть свой ник из ТОПа одной кнопкой.
*   **Поддержка медиа:** Бот пересылает фото и видео без раскрытия данных автора.

## 🛡 Анонимность и безопасность

1.  **Для пользователя:** Бот скрывает ваш профиль в чате с администратором. Однако для работы системы ответов и ТОПа бот сохраняет ваш ID и Username/Name в локальную базу данных.
2.  **Для администратора:** Вы не видите, кто вам пишет в интерфейсе Telegram. Это позволяет получать честную обратную связь.
3.  **Безопасность:** Токен бота не хранится в коде, а используется через переменные окружения.

## 🚀 Запуск бота

### 1. Подготовка
Убедитесь, что у вас установлен **Python 3.8+** и библиотека **aiogram**:
```bash
pip install aiogram
```

### 2. Настройка переменных
*   Вставьте свой `ID` в переменную `ADMIN_ID` внутри файла бота.
*   Бот берет токен из переменной окружения `BOT_TOKEN`.

### 3. Команды для запуска
**Windows (CMD):**
```cmd
set BOT_TOKEN=ваш_токен
python main.py
```

**Linux / macOS:**
```bash
export BOT_TOKEN="ваш_токен"
python main.py
```

## 🛠 Технологии
*   **Язык:** Python 3
*   **Библиотека:** aiogram 3.x
*   **База данных:** SQLite (feedback_private.db)
*   **FSM:** Aiogram Context
