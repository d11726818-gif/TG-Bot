# 🛡️ TG-Anonymous-Bot (Open Source Edition)

Официальный исходный код анонимного чат-бота. 

## 🎯 Миссия проекта
Этот бот создан для того, чтобы обеспечить **100% анонимную связь** между пользователями и администрацией канала. Мы верим в свободу слова и право на приватность.

## 🔒 Почему этот бот безопасен?
1. **Прозрачность:** Весь код открыт для аудита. Вы можете лично убедиться, что здесь нет функций сбора личных данных (имен, юзернеймов, телефонов).
2. **Анонимные тикеты:** Каждому обращению присваивается технический `Ticket-ID`. Администратор видит только его, чтобы иметь возможность ответить вам через бота.
3. **Отсутствие баз данных:** Ваши сообщения не сохраняются на сервере навсегда.

## 🛠️ Технологии
* **Язык:** Python 3.10+
* **Библиотека:** Aiogram 3.x (самая современная и быстрая база)

## 🚀 Как запустить самому?
Если вы хотите убедиться в работе кода лично:
1. Клонируйте репозиторий.
2. Установите зависимости: `pip install aiogram`
3. Впишите свой `TOKEN` и `ADMIN_ID` в файл `main.py`.
4. Запустите бота: `python main.py`
